# 무료 API 활용 기능 제안

한국 안과·성형외과 클리닉에 유용한 무료 API 기반 기능들입니다.

## 🎯 추천 기능 목록

### 1. 공휴일 체크 API (공공데이터포털) ⭐⭐⭐
**API**: [공공데이터포털 - 공휴일 정보](https://www.data.go.kr/)
- **무료**: 완전 무료
- **용도**: 예약 가능 날짜 필터링, 공휴일 자동 제외
- **기능**:
  - 예약 달력에서 공휴일 자동 표시
  - 공휴일 예약 불가 처리
  - 공휴일 안내 메시지 자동 추가

### 2. Kakao Map API ⭐⭐⭐
**API**: [Kakao Developers - 지도 API](https://developers.kakao.com/)
- **무료**: 월 300,000건 무료
- **용도**: 병원 위치 지도 표시, 길찾기
- **기능**:
  - 환자 페이지에 병원 위치 지도 표시
  - 예약 확인 메시지에 지도 링크 포함
  - 길찾기 기능

### 3. QR 코드 생성 (로컬 라이브러리) ⭐⭐
**라이브러리**: `qrcode` (npm 패키지)
- **무료**: 완전 무료
- **용도**: 예약 QR 코드, 쿠폰 QR 코드
- **기능**:
  - 예약 확인서 QR 코드 생성
  - 쿠폰 QR 코드 생성
  - 환자용 QR 코드 다운로드

### 4. 주소 검색 API (Kakao/Naver) ⭐⭐
**API**: [Kakao 주소 검색 API](https://developers.kakao.com/docs/latest/ko/local/dev-guide#search-by-keyword)
- **무료**: 월 300,000건 무료
- **용도**: 환자 주소 입력 시 자동완성
- **기능**:
  - 주소 검색 자동완성
  - 정확한 주소 입력
  - 우편번호 자동 입력

### 5. 날씨 API (OpenWeatherMap) ⭐
**API**: [OpenWeatherMap](https://openweathermap.org/api)
- **무료**: 일 1,000건 무료
- **용도**: 예약일 날씨 안내
- **기능**:
  - 예약일 날씨 예보 표시
  - 날씨 기반 안내 메시지 (예: "비 예보가 있으니 우산을 준비하세요")

### 6. 이메일 발송 API (Resend) ⭐⭐⭐
**API**: [Resend](https://resend.com/)
- **무료**: 월 3,000건 무료
- **용도**: 이메일 마케팅 추가 채널
- **기능**:
  - 예약 확인 이메일 발송
  - 마케팅 이메일 발송
  - 이메일 템플릿 관리

### 7. 공휴일 달력 API (공공데이터) ⭐⭐
**API**: [공공데이터포털 - 달력 정보](https://www.data.go.kr/)
- **무료**: 완전 무료
- **용도**: 예약 달력에 공휴일 표시
- **기능**:
  - 한국 공휴일 자동 표시
  - 예약 불가 날짜 자동 처리

## 🚀 구현 우선순위

2. **QR 코드 생성** - 예약 확인서에 유용
3. **Kakao Map API** - 병원 위치 표시

### Phase 3: 장기 구현 (낮은 가치)
6. **날씨 API** - 부가 기능

## 📋 각 기능별 상세 설명

### 1. 공휴일 체크 기능
```typescript
// 예약 가능 날짜 필터링
- 예약 달력에서 공휴일 자동 제외
- 공휴일 예약 시도 시 경고 메시지
- 공휴일 안내 템플릿 변수 추가 ({{holiday_info}})
```

### 2. Kakao Map 통합
```typescript
// 병원 위치 표시
- 환자 대시보드에 지도 위젯
- 예약 확인 메시지에 지도 링크
- 길찾기 버튼 제공
```

### 3. QR 코드 생성
```typescript
// 예약 QR 코드
- 예약 생성 시 QR 코드 자동 생성
- 환자에게 QR 코드 전송
- QR 코드로 예약 확인 가능
```

### 4. 주소 검색
```typescript
// 주소 자동완성
- 환자 등록 시 주소 검색
- Daum/Naver 주소 API 연동
- 정확한 주소 입력 보장
```

### 5. 날씨 정보
```typescript
// 날씨 기반 안내
- 예약일 날씨 예보 표시
- 날씨 기반 맞춤 메시지
- 외출 시 주의사항 안내
```

### 6. 이메일 발송
```typescript
// 이메일 마케팅
- 예약 확인 이메일
- 마케팅 이메일 발송
- 이메일 템플릿 관리
```

## 🔧 구현 방법

각 기능은 독립적으로 구현 가능하며, 점진적으로 추가할 수 있습니다.

1. **공휴일 API**: 예약 페이지에 통합
2. **QR 코드**: 예약 생성 시 자동 생성
3. **지도 API**: 설정 페이지에서 병원 위치 설정 후 표시
4. **주소 검색**: 환자 등록 폼에 통합
5. **이메일 API**: 템플릿에 이메일 채널 추가
6. **날씨 API**: 예약 상세 페이지에 표시

## 💡 추가 아이디어

- **공휴일 자동 안내**: 공휴일 전날 자동 안내 메시지
- **지도 기반 거리 계산**: 환자 거리별 맞춤 메시지
- **QR 코드 기반 체크인**: 현장에서 QR 코드 스캔으로 체크인
- **날씨 기반 예약 추천**: 날씨 좋은 날 예약 유도
