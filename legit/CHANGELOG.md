- **날짜**: 최근
- **변경 사항**:

  - 실시간 입력 검증 및 에러 메시지
  - 발송 성공 시 입력 필드 자동 초기화
- **파일**:
  - `app/[locale]/dashboard/page.tsx` - 전화번호 입력 UI 추가
  - `lib/phone-validation.ts` - 전화번호 검증 유틸리티 추가

- **날짜**: 최근
- **변경 사항**:

- **파일**:
  - `lib/env-validation.ts` - 환경 변수 검증 유틸리티 추가

  - `app/layout.tsx` - 서버 사이드 검증 추가
